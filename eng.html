1 Steam Tags Analysis

My team performed a statistical analysis on video games, using the tags and reviews data we scraped from Steam, one of the largest Digital Distribution Platforms. My first role in this project was to make a web crawler in R 3.5.1. I used rvest 0.3.2 package for crawling, and XML 3.98 package to refine the data into a nice .csv file. I calculated various statistical indicators such as eigenvector centrality and pagerank using Gephi 0.9.2. Using the modularity my teammate obtained using the Louvain Method, I visualized our data as a network on Gephi and as an alluvial diagram using R.

2 Steam Review Prediction Model

For this personal project, I obtained a already cleaned data of all video games on Steam as .json from SteamSpy API. I transformed it into a .csv so I can train in using Tensorflow. This data undergoes ConvNet, max pooling, and batch normalization multiple times and finally outputs a review prediction through a RELU function. It was written in Python 3.6.7, and many useful libraries such as Numpy, Pandas, and Tensorflow were used. Tensorflow-GPU 1.12.0 was executed under Windows 10 and GTX1080, along with Nvidia CUDA 10.0.130, Nvidia cuDNN 7.4.1, and Microsoft Visual C++ 2015 Update 3.

3 PlayDB Crawler

This Rscript scrapes titles, premiere dates, closing dates, casts, and reviews of every show from playdb.com, the largest database for concerts, musicals, festivals, and more held in Korea. It was written in R 3.5.1 using rvest 0.3.2 package, and the data refinement was doen using XML 3.98 and plyr 1.8.4 packages.


4 An Interactive Network of KAIST Professors

This is a visualization of all KAIST professors' department choices as they've gone from undergrad to masters to Ph.D. The original data is of 2015, provided by the academic administration of KAIST. On R 3.5.1, I removed all duplicates and empty rows and standardized attribute values. The data then went through de-identification for privacy. All new duplicates were added up to a new row called edge_weight. After normalizing, I visualized the data using Gephi 0.9.2 and exported the result as a HTML5 interactive network via SigmaExporter 0.9.0 plug-in. I also edited index.html, style.css and config.jss for better UX and readability.

5 Handwritten Digits Recognizer

This is a deep learning model that recognizes handwritten digits. First, I got my train and test data from MNIST database. I augmented my train data using ImageDataGenerator of the keras library, and then I normalized it on Python 3.6.7 using NumPy. The training data then went through a series of convolutional networks, max pooling and batch normalization. The result was put in RELU and Softmax to reach conclusion. After training, the test data was fed into the model, and the evaluation resulted in 99.99% accuracy.

6 Twitter Word Cloud

This is a Rscript for creating a word cloud from twitter hashtags. This script takes a hashtag of the user's choice as an input, and crawls Twitter for N tweets with that particular hashtag. The scraped tweets are turned into a word cloud, with their sizes corresponding to their frequencies. It was written in R.3.5.1 using rtweet package and my Twitter API as a Twitter developer. The data was processed using tidytext, dplyr, stringr, and wordcloud2 packages.
